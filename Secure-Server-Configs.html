Secure Server Configurations

Linux Hardening

  • SSH Config: Bloqueo de acceso root, claves SSH, restricción de puertos.
  • sysctl Hardening: Protección del kernel contra ataques DoS y spoofing.
  • UFW Firewall: Reglas para un firewall seguro.

Windows Hardening

  • Windows Defender: Configuración reforzada con PowerShell.
  • Políticas de Seguridad: Registro para reforzar GPOs.

Web Server Hardening

  • Apache: Headers de seguridad, SSL/TLS, restricción de métodos inseguros.
  • Nginx: Configuración optimizada para seguridad y rendimiento.
  • IIS: Configuración de seguridad para servidores Microsoft.
